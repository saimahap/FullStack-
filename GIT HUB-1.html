Learning Git & GitHub Step by Step

Step 1: Start a New Git Project

First, we create a new Git repository using:

git init

Step 2: Save Our Work

Now we add files and save them with a commit:

git add .
git commit -m "My first commit"

Step 3: Upload to GitHub

We send our project to GitHub using:

git push origin main

Step 4: Create a New Branch

To work on a new feature safely:

git checkout -b feature

Step 5: Merge the Branch

After finishing the feature, we merge it back:

git merge feature

Step 6: Update Our Project

To get the latest changes from GitHub:

git pull origin main
